entity-manifest

A tiny open standard that lets websites declare who they are and which hubs they belong to.

Think: robots.txt, but for entities and relationships.

Why this exists

AI systems increasingly select and summarize the web. Without explicit signals, they often guess: identity, affiliation, or official status.

entity-manifest prevents AI misinterpretation.

How it works

Publish a small JSON file at: 

/\.well-known/entity.json

This file describes the entity behind a website and its declared relationships.

Minimal example

{
  "spec": "entity-manifest/1.0",
  "entity": {
    "name": "Boulangerie Dupont",
    "type": "local-business",
    "url": "https://boulangerie-dupont.fr"
  },
  "belongsTo": [
    "https://viroflay.fr/.well-known/hub.json"
  ]
}

Without this file, AI agents should avoid inferring identity or affiliation.

No scoring. No ranking. No central authority.
